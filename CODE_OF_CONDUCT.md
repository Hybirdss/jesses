# Code of Conduct

This project adopts the **Contributor Covenant version 2.1** verbatim and without modification.

- Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Translations: <https://www.contributor-covenant.org/translations>
- License: CC BY 4.0 (attribution: Contributor Covenant)

The Covenant applies to all project spaces: this repository, its issue tracker, its pull requests, its discussions, its release threads, and any in-person or virtual events organized under the project's name.

## Enforcement contact

Reports to the maintainer listed in `MAINTAINERS.md`, via the email address published there. Reports are held in confidence and acknowledged within 72 hours.

During the v0.1 build phase the project has a single maintainer, which means the enforcement channel is not independent of the person running the project. This limitation is recorded honestly. After the Commonhaus transition (`GOVERNANCE.md` §Transition), enforcement moves to the foundation's conduct committee, which is independent of the maintainer.

## Scope

This Code applies to everyone who participates in the project — contributors, maintainers, issue reporters, release reviewers, downstream implementers, integration partners, and speakers representing the project at conferences or online events. Participation is conditional on adherence; persistent violations result in loss of participation privileges in the project's spaces.
