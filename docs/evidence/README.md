# Evidence Archive

Every dated claim in [`WHY.md`](../../WHY.md) and other public documents cites a public source. This directory preserves the **substance** of those sources as local summaries, so the claim survives URL rot, platform deletion, or vendor revision.

This is not a replacement for the original. It is a safety net. When the original is alive and reachable, read the original.

## Archive discipline

- Each evidence file has a **date** in the filename (`YYYY-MM-DD`).
- Each file contains: the original URL, an optional Wayback Machine archive URL, a 1-paragraph summary in our words, and a small number of **direct quoted excerpts** (fair-use: short, attributed, for critical commentary — no full-article reproduction).
- If an original source rewrites or retracts, the local file adds an **addendum** dated and attributed; the original summary is not deleted.
- All sources are linked back from `WHY.md` with the inline `[label](url)` pointing at the **original**, and a footnote-style pointer to the archive file for readers who need the redundancy.

## Refresh cadence

Quarterly. Walk each URL, confirm still live, update status. If any goes dark, archive the Wayback URL front-and-center and note the outage date.

## Contributions

If you find additional public dated sources that materially support or contradict the WHY.md thesis, a PR adding a new file here is welcomed. Include:
- Date of source publication (not date you discovered it)
- Original URL
- Wayback or other archive URL
- A 3–5 sentence summary in your own words
- Any quotation under 75 words total, attributed, relevant
- A sentence on how this evidence changes or reinforces a WHY.md claim
